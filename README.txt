This module converts HTML into plain text.

The kind of thing you might want to do if you're sending the contents of a HTML page as a plain text email.

H1, H2, H3 text is converted to uppercase, an asterix is added to list items, and links can be listed at the bottom of the text. 

Send comments to Robert Castelo at development@cortextcommunications.com
